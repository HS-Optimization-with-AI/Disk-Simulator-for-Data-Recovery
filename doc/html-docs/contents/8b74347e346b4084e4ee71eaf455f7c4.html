CreateFile

UMLOperation
Disk Simulator for Data recovery :: Model :: Memory :: CreateFile

Description

Creates new file with the given linking factor and of the given size. - Form blocklist of the given size with num_blocks number of max priority score blocks from unused heap (check for memory full) - Add these blocks to used blocks - Add this file to fileList - Call createFile function of the file with blocklist and linking factor - Call updateHeap() - return file index to IO gen

Parameters

Direction Name Type Description
in int linking_factor
in int num_blocks

Creates a new file with given linking factor and allocates the number of blocks as input. - Forms a blockList with the popping num_blocks number of max priority score blocks from unused heap. (check memory full) - Uses createFile constructor to allocate these blocks to form a new file with input linking factor. - Adds this file to the fileList - Return index of this file for the IO gen
