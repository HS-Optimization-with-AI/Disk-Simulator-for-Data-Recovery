public performance(Memory mem)

Calculate RR and SLM of the given memory - Initialize rr and slm to 0 - Iterate through all files of the memory - For used files : slm += file.slm() - For unused files : rr += file.rr() - performance is defined as : rr/numunused - weight*slm/numused - return performance
