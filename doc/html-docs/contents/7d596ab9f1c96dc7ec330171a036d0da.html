public generateIO(int memutil)

Generate a random operation based on the memory utilization parameters - Randomly choose from create, delete and read/write - The operation / transaction chosen should obey that the memutil is a probabilistic distribution from 80 to 90 %. Not create when 90, no delete when 80. - For create, choose randomly lf 0 or 10 with probability 0.2 and 0.8. Store returned file index in fileIndices. - For delete randomly choose from the fileIndices. Assert that the file is in used state. remove this index from fileIndices. - For read/write, choose randomly from the fileIndices. - If memory utilization is stabilized to range 80 to 90 then set stable to true - Return the action
