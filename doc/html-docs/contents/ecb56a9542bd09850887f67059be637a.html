Operations

Visibility Name Description
public Memory(int w, int h)

Constructor of memory Sets memory width and height Initializes all blocks in memory Sets their i and j Initializes fileList, used and unused attributes
public CreateFile(int linking_factor, int num_blocks)

Creates new file with the given linking factor and of the given size. - Form blocklist of the given size with num_blocks number of max priority score blocks from unused heap (check for memory full) - Add these blocks to used blocks - Add this file to fileList - Call createFile function of the file with blocklist and linking factor - Call updateHeap() - return file index to IO gen
public DeleteFile(int fileIndex)

Deletes the file corresponding to the fileIndex. Asserts that the file is used. Call the deleteFile() method of the file and insert the blocks of the file in the unused heap. Delete these blocks from the used list. Call updateHeap()
public ReadWriteFile(int fileIndex)

Read/Write operation for the file corresponding to the index provided

Asserts that the file is used.

Call updateHeap()
public updateSF()

Updates Spatial Factors of all blocks by taking an average of the pf of the neighbouring blocks by calling the setSF() method
public refresh()

Updates the priority scores of all blocks in memory. Calls the updateSF() function. Calls the updatepf(lambda, sigma, rho, mu) function for all blocks in memory. Percolate all heap nodes for unused blocks in the heap based on their priority scores. Update memutil = num of blocks used / (width*height)
public updateParams(int l, int s, int r, int m)

Update lamda, sigma, rho, mu of memory
